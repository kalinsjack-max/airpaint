# Air Drawing App (Single HTML File) Save this as `index.html` and open it in a browser. ```html Air Drawing App

Air Drawing

Waiting for hand...
☝ One Finger = Draw
✋ Open Palm = Erase
✊ Closed Fist = Pause
``` ## Features * MediaPipe hand tracking * Neon glowing drawing trail * One finger gesture to draw * Open palm gesture to erase * Closed fist gesture to pause * Brush color picker * Brush thickness slider * Dark modern UI * Webcam preview
